# Advent of Code

## Repository structure

To run a specific binary, run:

```sh
cargo run -p year-{year} --bin day-{day}
```

Example:

```sh
cargo run -p year-2024 --bin day-01
```

## Create new day

```sh
cargo run -p cli
```

this will inquire `year` and `day`

Example:

> Year:  2024
> Day:  3

and generate the scaffold.
